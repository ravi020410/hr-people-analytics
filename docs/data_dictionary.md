# Data Dictionary

| Field | Description |
|---|---|
| EmployeeID | Unique employee identifier. |
| Department | Employee department. |
| JobRole | Employee role title. |
| Age | Employee age. |
| Gender | Synthetic demographic group. |
| TenureYears | Employee tenure in years. |
| PerformanceScore | Performance rating from 1 to 4. |
| ActiveStatus | 1 for active employee, 0 for voluntary exit. |
| EmployeeSalary | Synthetic current salary. |
| MarketBenchmarkSalary | Benchmark salary for the same role. |
| SalaryDeviationPct | Employee salary deviation from benchmark. |
| JobSatisfaction | Survey score from 1 to 5. |
| WorkLifeBalance | Survey score from 1 to 5. |
| PrimaryExitReason | Exit reason for exited employees; active employees marked separately. |
